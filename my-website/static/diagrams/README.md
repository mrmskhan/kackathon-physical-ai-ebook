# Diagrams Directory

This directory contains exported diagrams and architecture visuals for the e-book.

## Structure

- `module-01-ros2/` - ROS 2 Foundation diagrams
- `module-02-simulation/` - Simulation architecture diagrams
- `module-03-isaac-ai/` - Isaac Sim workflow diagrams
- `module-04-vla/` - VLA pipeline diagrams
- `module-05-hardware/` - Hardware setup diagrams
- `capstone/` - Capstone project diagrams

## File Naming Convention

Use descriptive names: `module-XX-topic-description.png`

Examples:
- `module-01-ros2-pubsub-architecture.png`
- `module-04-vla-pipeline-overview.png`

## Image Guidelines

- **Format**: PNG for screenshots, SVG for diagrams when possible
- **Resolution**: Minimum 1200px width for readability
- **Compression**: Use optimized PNG to reduce file size
- **Alt Text**: Always include descriptive alt text in markdown
